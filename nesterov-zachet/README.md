## Описание

Это небольшая работа по ОАИП.  
На странице загружаются посты с API и выводятся в виде карточек.

Также есть форма, через которую можно создать новый пост.  
Форма отправляет POST-запрос без перезагрузки страницы.

## Что сделано

- получение постов с API;
- вывод постов на страницу;
- форма для создания поста;
- POST-запрос через fetch;
- обработка ошибок через try-catch.

## API

Получение постов:

GET https://jsonplaceholder.typicode.com/posts

Description

This is a small OAIP work.
The page loads posts from the API and displays them as cards.

There is also a form for creating a new post.
The form sends a POST request without reloading the page.

What is done
loading posts from API;
displaying posts on the page;
form for creating a post;
POST request using fetch;
error handling with try-catch.
API

Get posts:

GET https://jsonplaceholder.typicode.com/posts

Create post:

POST https://jsonplaceholder.typicode.com/posts
Run

Open the index.html file in a browser.

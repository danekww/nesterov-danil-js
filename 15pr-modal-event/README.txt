[15пр] Работа с объектом Event на примере модального окна
Выполнил: Нестеров Данил Юрьевич

Как открыть:
1. Откройте папку 15pr-modal-event-nesterov в Visual Studio Code.
2. Откройте файл index.html.
3. Запустите через Live Server или откройте index.html в браузере.

Что реализовано:
1. Модалка-подтверждение:
   - кнопка «Удалить профиль» открывает модальное окно;
   - кнопка «Отмена» закрывает окно;
   - кнопка «Да, удалить» выводит в консоль «Профиль удален» и закрывает окно;
   - закрытие реализовано только по кнопкам.

2. Галерея картинок:
   - на странице есть три маленькие картинки;
   - при клике на картинку открывается модальное окно;
   - картинка внутри модального окна получает src той картинки, по которой кликнули;
   - используется event.target.

3. Умная форма:
   - есть модалка с формой подписки;
   - окно закрывается при клике на оверлей;
   - используется stopPropagation;
   - если Email пустой, поле подсвечивается красным;
   - если Email введен, в консоль выводится «Успех!» и окно закрывается.

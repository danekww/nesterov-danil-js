Новые вариации счётчиков
Выполнил: Нестеров Данил Юрьевич

Как открыть:
1. Распакуйте архив.
2. Откройте папку counter_variations_nesterov в Visual Studio Code.
3. Откройте файл index.html.
4. Запустите его через Live Server или просто откройте index.html в браузере.

Файлы:
- index.html — разметка страницы.
- style.css — оформление.
- script.js — логика на JavaScript.

Что реализовано:
1. Счётчик-настройщик: значение меняется на число из input.
2. Светофор: положительное значение зелёное, отрицательное красное, ноль чёрный.
3. Уровень заряда: счётчик от 0 до 100 и статус заряда.

[16пр] querySelectorAll и итерация объектов
Выполнил: Нестеров Данил Юрьевич

Задание:
Интерактивный список дел (To-Do List).

Что реализовано:
1. Через querySelectorAll найдены все элементы с классом .todo__item.
2. С помощью forEach перебрана коллекция задач.
3. Внутри каждой задачи найдена кнопка с классом .todo__btn.
4. На каждую кнопку добавлен обработчик onclick.
5. При клике переключается класс done у родительского элемента .todo__item.

Как открыть:
1. Откройте папку 16pr-todo-list в Visual Studio Code.
2. Откройте файл index.html.
3. Запустите через Live Server или откройте index.html в браузере.
